## Overview

A lightweight business diagnostic tool that helps founders and operators determine whether their business is truly creating value or simply generating activity.

The tool calculates Return on Invested Capital (ROIC) using simple monthly inputs and provides instant interpretation.

---

## The Problem

Many small and mid-sized businesses track revenue and profit but lack visibility into capital efficiency.

Revenue growth does not always equal value creation.

Without understanding return on invested capital, businesses may scale inefficiency.

---

## The Solution

This tool provides a simple, structured way to:

- Estimate Return on Invested Capital
- Understand capital efficiency
- Evaluate whether growth creates or destroys value
- Improve strategic decision-making

---

## Key Features

- Simple 2-minute input flow
- Clear ROIC calculation logic
- Instant interpretation
- Strategic guidance based on result
- Instant summary sent directly to email (no signup required)
